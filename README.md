# Leaseweb Price Calculator

A lightweight web-based price calculator designed to simplify Leaseweb instance and storage pricing.  
This tool provides a clean interface for users to explore CPU, GPU, storage, and traffic pricing across multiple regions.

## Features

- ✔ Real-time pricing calculation  
- ✔ CPU instance pricing  
- ✔ GPU instance pricing  
- ✔ Central storage & Local NVMe storage  
- ✔ Outgoing traffic pricing tiers  
- ✔ Region-based price selection  
- ✔ Fully client-side (no backend required)  
- ✔ JSON-driven pricing (easy to maintain)
